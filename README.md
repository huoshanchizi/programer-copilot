# Programer Copilot
拥抱新技术，更高效的创造服务帮助客户解决痛点。

## 为什么制作这个？
有一个想法之后，可以快速提供一些参考的技术方案，虽然方案可能不完整甚至不对，但如此之快的反馈，会促使你更快的迭代思考。
千里之行，始于足下。

## 程序员会不会被机器替代？
如果只是纯粹按照产品经理设计好的各个小功能点，写写代码，很有可能会逐步被淘汰。就像翻译一样，现在机器翻译已经非常的强大。
把功能拆解得足够细，GPT 也能生成代码。 后面就是集成、测试。
